Ejercicio 1: ICDT#

Enunciado#

Se desea enfriar un flujo de tolueno (\(0.48~\text{kg/s}\)) hasta alcanzar los \(75~\text{°C}\). Para esto, se utiliza un flujo de agua (\(0.55~\text{kg/s}\)) a contracorrientes que pasa por el ánulo de un intercambiador de doble tubo. Al inicio de la operación, el agua se encuentra a \(10~\text{°C}\) y termina saliendo del proceso a \(20~\text{°C}\). El tubo es de acero (\(k_p = 50.2~\text{W/m}\cdot\text{K}\)) y considere que su diámetro interno del tubo es \(D_i=15~\text{cm}\), que el diámetro externo del tubo es \(D_{i,e}=15.5~\text{cm}\) y que el díametro interno del ánulo es \(D_a= 25~\text{cm}\).

Para las condiciones descritas en el enunciado, las propiedades termofísicas del tolueno y el agua son las siguientes:

Encuentre lo siguiente:

  1. La temperatura de entrada del tolueno.

  2. Los coeficientes de transferencia de calor para fluido que circula por el tubo y por el ánulo.

  3. Si el coeficiente de transferencia global total (\(U_T\)) es \(28.43~\text{W/m}^2\cdot\text{K}\), encuentre el factor de ensuciamiento.

Solución#

import handcalcs.render
from handcalcs import handcalc
from numpy import pi, log

Inciso 1#

Para encontrar la temperatura de entrada del tolueno, podemos igualar los valores de \(q\):

\[ q = \dot{m}_hc_{P,h}(T_{h,e}-T_{h,s}) = \dot{m}_cc_{P,c}(T_{c,s}-T_{c,e}) \]

Para este ejercicio, el fluido caliente (tolueno) con subíndice \(h\) pasa por el tubo, mientras que el fluido de enfriamiento (agua) con subíndice \(c\) pasa por el ánulo. Del enunciado recuperamos todas las variables relevantes.

%%render params sci_not
m_c = 0.55 #kg/s
T_ce = 10+273.15 #K
T_cs = 20+273.15 #K
mu_c = 8.994e-4 #Pa$\cdot$s
c_Pc = 4181.95 #J/kg$\cdot$K
k_c = 0.6062 #W/m$\cdot$K
m_h = 0.48 #kg/s
T_hs = 75+273.15 #K
mu_h = 3.003e-4 #Pa$\cdot$s
c_Ph = 1917.25 #J/kg$\cdot$K
k_h = 0.11329 #W/m$\cdot$K
\[\begin{split} \begin{aligned} m_{c} &= 5.500 \times 10 ^ {-1} \; \;\textrm{(kg/s)} &T_{ce} &= 2.831 \times 10 ^ {2} \; \;\textrm{(K)} &T_{cs} &= 2.931 \times 10 ^ {2} \; \;\textrm{(K)} \\[10pt] \mu_{c} &= 8.994 \times 10 ^ {-4} \; \;\textrm{(Pa$\cdot$s)} &c_{Pc} &= 4.182 \times 10 ^ {3} \; \;\textrm{(J/kg$\cdot$K)} &k_{c} &= 6.062 \times 10 ^ {-1} \; \;\textrm{(W/m$\cdot$K)} \\[10pt] m_{h} &= 4.800 \times 10 ^ {-1} \; \;\textrm{(kg/s)} &T_{hs} &= 3.481 \times 10 ^ {2} \; \;\textrm{(K)} &\mu_{h} &= 3.003 \times 10 ^ {-4} \; \;\textrm{(Pa$\cdot$s)} \\[10pt] c_{Ph} &= 1.917 \times 10 ^ {3} \; \;\textrm{(J/kg$\cdot$K)} &k_{h} &= 1.133 \times 10 ^ {-1} \; \;\textrm{(W/m$\cdot$K)} \end{aligned} \end{split}\]
%%render long
q = m_c*c_Pc*(T_cs-T_ce) #J/s
T_he = q/(m_h*c_Ph)+T_hs #K
\[\begin{split} \begin{aligned} q &= m_{c} \cdot c_{Pc} \cdot \left( T_{cs} - T_{ce} \right) \\&= 0.550 \cdot 4181.950 \cdot \left( 293.150 - 283.150 \right) \\&= 23000.725 \; \;\textrm{(J/s)}\\[10pt] \\[10pt] T_{he} &= \frac{ q }{ m_{h} \cdot c_{Ph} } + T_{hs} \\&= \frac{ 23000.725 }{ 0.480 \cdot 1917.250 } + 348.150 \\&= 373.143 \; \;\textrm{(K)}\\[10pt] \end{aligned} \end{split}\]

Inciso 2#

Para calcular los coeficientes de transferencia de calor vienen dados por el número de Nusselt.

\[ Nu_D = \frac{hD_{térmico}}{k} \]

El cual puede ser despejado de distintas correlaciones dependiendo del tipo de fluido. Por lo que es recomendable empezar calculando el número de Reynolds para discriminar el tipo de flujo.

\[ Re_D = \frac{D_{térmico}\cdot\dot{m}}{A_t\cdot\mu} \]

Por un lado, el diámetro térmico equivalente para el flujo que pasa por el tubo será equivalente al diámetro interno de la tubería. Por otro lado, el diámetro térmico equivalente para el flujo que pasa por el ánulo, será el siguiente:

\[ D_{térmico} = \frac{D_{a}^2-D_{i,e}^2}{D_{i,e}} \]

Luego, podemos calcular los números de Prandtl y de Reynolds para ambos flujos.

%%render params
D_i = 15/100 #m
D_ie = 15.5/100 #m
D_a = 25/100 #m
k_p = 50.2 #W/m$\cdot$K
Pr_h = (mu_h*c_Ph)/k_h
Pr_c = (mu_c*c_Pc)/k_c
\[\begin{split} \begin{aligned} D_{i} &= 0.150 \; \;\textrm{(m)} &D_{ie} &= 0.155 \; \;\textrm{(m)} &D_{a} &= 0.250 \; \;\textrm{(m)} \\[10pt] k_{p} &= 50.200 \; \;\textrm{(W/m$\cdot$K)} &\mathrm{Pr}_{h} &= 5.082 \; &\mathrm{Pr}_{c} &= 6.205 \; \\[10pt] \end{aligned} \end{split}\]
%%render long
Re_Dh = (D_i*m_h)/(pi*(D_i/2)**2*mu_h)
D_t = (D_a**2-D_ie**2)/D_ie**2 #m
Re_Dc = (D_t*m_c)/(pi*((D_a-D_ie)/2)**2*mu_c)
\[\begin{split} \begin{aligned} \mathrm{Re}_{Dh} &= \frac{ D_{i} \cdot m_{h} }{ \pi \cdot \left( \frac{ D_{i} }{ 2 } \right) ^{ 2 } \cdot \mu_{h} } \\&= \frac{ 0.150 \cdot 0.480 }{ 3.142 \cdot \left( \frac{ 0.150 }{ 2 } \right) ^{ 2 } \cdot 0.000 } \\&= 13567.654 \\[10pt] \\[10pt] D_{t} &= \frac{ \left( D_{a} \right) ^{ 2 } - \left( D_{ie} \right) ^{ 2 } }{ \left( D_{ie} \right) ^{ 2 } } \\&= \frac{ \left( 0.250 \right) ^{ 2 } - \left( 0.155 \right) ^{ 2 } }{ \left( 0.155 \right) ^{ 2 } } \\&= 1.601 \; \;\textrm{(m)}\\[10pt] \\[10pt] \mathrm{Re}_{Dc} &= \frac{ D_{t} \cdot m_{c} }{ \pi \cdot \left( \frac{ D_{a} - D_{ie} }{ 2 } \right) ^{ 2 } \cdot \mu_{c} } \\&= \frac{ 1.601 \cdot 0.550 }{ 3.142 \cdot \left( \frac{ 0.250 - 0.155 }{ 2 } \right) ^{ 2 } \cdot 0.001 } \\&= 138161.788 \\[10pt] \end{aligned} \end{split}\]

Notamos que ambos flujos son turbulentos, y por lo tanto, se cosidera que la mejor ecuación para encontrar el número de Nusselt será la relación de Dittus-Boelter.

%%render long
NuD_h = 0.023*Re_Dh**0.8*Pr_h**0.3
NuD_c = 0.023*Re_Dc**0.8*Pr_c**0.4
\[\begin{split} \begin{aligned} \mathrm{NuD}_{h} &= 0.023 \cdot \left( \mathrm{Re}_{Dh} \right) ^{ 0.8 } \cdot \left( \mathrm{Pr}_{h} \right) ^{ 0.3 } \\&= 0.023 \cdot \left( 13567.654 \right) ^{ 0.8 } \cdot \left( 5.082 \right) ^{ 0.3 } \\&= 75.778 \\[10pt] \\[10pt] \mathrm{NuD}_{c} &= 0.023 \cdot \left( \mathrm{Re}_{Dc} \right) ^{ 0.8 } \cdot \left( \mathrm{Pr}_{c} \right) ^{ 0.4 } \\&= 0.023 \cdot \left( 138161.788 \right) ^{ 0.8 } \cdot \left( 6.205 \right) ^{ 0.4 } \\&= 618.194 \\[10pt] \end{aligned} \end{split}\]

Con lo cual se puede despejar el coeficiente de transferencia de calor para ambos fluidos.

%%render long
h_h = (NuD_h*k_h)/(D_i) #W/m$^2\cdot$K
h_c = (NuD_c*k_c)/(D_t) #W/m$^2\cdot$K
\[\begin{split} \begin{aligned} h_{h} &= \frac{ \mathrm{NuD}_{h} \cdot k_{h} }{ D_{i} } \\&= \frac{ 75.778 \cdot 0.113 }{ 0.150 } \\&= 57.233 \; \;\textrm{(W/m$^2\cdot$K)}\\[10pt] \\[10pt] h_{c} &= \frac{ \mathrm{NuD}_{c} \cdot k_{c} }{ D_{t} } \\&= \frac{ 618.194 \cdot 0.606 }{ 1.601 } \\&= 234.005 \; \;\textrm{(W/m$^2\cdot$K)}\\[10pt] \end{aligned} \end{split}\]

Inciso 3#

Primero, calculamos el coeficiente global de transferencia de calor para un intercambiador de doble tubo sin ensuciamiento.

%%render long
U = (D_ie/(D_i*h_h) + D_ie/(2*k_p)*log(D_ie/D_i) + 1/h_c)**-1 #W/m$^2\cdot$K
\[\begin{split} \begin{aligned} U &= \left( \frac{ D_{ie} }{ D_{i} \cdot h_{h} } + \frac{ D_{ie} }{ 2 \cdot k_{p} } \cdot \ln \left( \frac{ D_{ie} }{ D_{i} } \right) + \frac{ 1 }{ h_{c} } \right) ^{ -1 } \\&= \left( \frac{ 0.155 }{ 0.150 \cdot 57.233 } + \frac{ 0.155 }{ 2 \cdot 50.200 } \cdot \ln \left( \frac{ 0.155 }{ 0.150 } \right) + \frac{ 1 }{ 234.005 } \right) ^{ -1 } \\&= 44.685 \; \;\textrm{(W/m$^2\cdot$K)}\\[10pt] \end{aligned} \end{split}\]

Luego podemos despejar el factor de ensuciamiento.

\[ \frac{1}{U_T} = \frac{1}{U} + R_f \]
%%render long
U_T = 28.43 #W/m$^2\cdot$K
R_f = 1/U_T - 1/U #m$^2\cdot$K/W
\[\begin{split} \begin{aligned} U_{T} &= 28.430 \; \;\textrm{(W/m$^2\cdot$K)} \\[10pt] R_{f} &= \frac{ 1 }{ U_{T} } - \frac{ 1 }{ U } \\&= \frac{ 1 }{ 28.430 } - \frac{ 1 }{ 44.685 } \\&= 0.013 \; \;\textrm{(m$^2\cdot$K/W)}\\[10pt] \end{aligned} \end{split}\]

Finalmente, obtenemos que el factor de ensuciamiento es de \(R_f = 0.013~\text{m}^2\cdot\text{K/W}\).
